Browser Based IDE

MERN Stack | AWS

Introduction

This is an IDE that can be accessed from the browser, no need for installation, save your code in the cloud, access it from any computer, and code on the go.

Frontend

I developed the frontend using React.js and it was a considerable challenge to develop as it included a lot of complex componenets that had to be developed from scratch. Some of the most complex parts of the frontend were the file directory and the object structure used to contain the code files and link the directory to the editor.

Backend

For the backend APIs I chose to go with Node and Express and the DBMS I used is MongoDB. Figuring out how to maintain state for shell connections and link them to the frontend was a big challenge. I decided to use a websocket connection to achieve this, as the websockets maintain state and hold the connection open.

Deployment

AWS was used for hosting, as it's a one step solution for hosting a full stack app and provides a lot of other useful services in one bundle

The frontend was hosted using static site hosting on S3 Buckets, and distributed using CloudFront CDN.

The backend was hosted on EC2 servers with an AWS Load Balancer sitting in front of it for scalability, the userfiles were stored in S3 and AWS CloudWatch was used to moniter the state of the servers and to get notified in case anything goes down.

Tech Stack

React.js / Node.js / Express.js / MongoDB / AWS - S3 | CloudFront | EC2 | LoadBalancer | CloudWatch
