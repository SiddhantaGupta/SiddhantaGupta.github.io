Introduction

A python module to analyze videos/images and gather data of human pose and hand positioning at different stages of an exercise.

Contribution

  • Implemented Image and Video processing to calculate angles of joints in human body using Mediapipe's holistic solution for pose and hand.
  • Developed AWS lambda to transcode vidoes to m3u8 streams using AWS MediaConvert for storage.
  • Developed lambda function for splitting videos using MoviePy and ffmpeg.
  • Developed AWS lambda to compress images for storage using Python Imaging Library (PIL).
  • Deployed the lambda functions as a lambda container using Docker, AWS ECR and AWS lambda.
