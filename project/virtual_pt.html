Virtual PT

Python | Mediapipe | Docker | AWS

Introduction

A python module to analyze videos/images and gather data of human pose and hand positioning at different stages of an exercise.

Module

Developed a python module using Google's Mediapipe library to gather data of all the angles for the body and hand from a given video/image to calculate angles of all joints during different stages of the exercise.

Deployment

Deployed docker image on AWS ECR and as an AWS Lambda container using the aws cli. The module had to be dockerized and deployed as a container as the module size was bigger than the base 50MB limit of lambda. lambda containers have a limit of 10GB.

Tech Stack

Python / Mediapipe / Docker / AWS - ECR | Lambda
