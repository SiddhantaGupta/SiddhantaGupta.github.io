Introduction

A lightweight express.js boilerplate built with modern tech for your next project. I built this to get up and running quickly whenever prototyping or starting a new project. This boilerplate is kept lightweight and barebones on purpose so that it can be used for any requirements but still gives the basic boilerplate and structure that could save days worth of time.
