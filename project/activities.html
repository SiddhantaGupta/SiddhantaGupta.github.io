Activities Web App

React.js

Introduction

This is a frontend only single page application that consumes the Bored API and suggests people who are bored, random fun activites that they can partake in. It allows people to search through various activities using advanced filters to get suggestions that are relevant to them and their mood. Users can save their favourite activities for later viewing in their browsers local storage.
