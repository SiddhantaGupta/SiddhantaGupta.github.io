Online Bookstore

Django | Python | sqlite3 | Bootstrap

Introduction

This is an online bookstore where book enthusiasts can buy the right book for them using filters and discuss their favourite books with the community. This is an e-commerce website with the basic functionality that is common to websites of this nature such the ability for users to create an account and save relevant data - their contact information and addresses, ability for users to search through the website using text as well as filter search, leaving a review, buy products and checkout their purchase history.
