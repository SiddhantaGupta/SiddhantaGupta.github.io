Introduction

Frontend for a service where scientists can log data gathered after doing research on animals.

Frontend

The frontend was to be ported over to web from an existing desktop client. The project had a complex ui requirement and needed heavy data management. Had to implement various screens that involved displaying a lot of data in an organized and intuitive manner to make it easier for the clients to make sense of it. I contributed to the project by adding various components and fixed a lot of existing bugs and made ui improvements.
