Introduction

This script creates convenient hotkeys for emulating i3wm style workspaces in Windows 11. I forked this to better mirror the mapping I use on linux (with i3wm), and it's always annoyed me that Windows does not have better hotkey support for this feature (for instance, there's no way to go directly to a desktop by number).
